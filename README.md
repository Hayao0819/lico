## Lico - シンプルで軽量なDotfilesマネージャー

Licoはシンプルで軽量なDotfilesマネージャーです。全てのファイルをGitで管理し、自由なファイル構造を維持できます。

このツールは[ssh0/dot](https://github.com/ssh0/dot)に影響を受けて開発されています。
